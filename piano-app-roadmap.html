‚ö° Technical Recommendations

Audio Library: Use Tone.js for precise timing and scheduling (Web Audio API wrapper)
Pitch Detection: Pitchfinder.js with autocorrelation algorithm for microphone input
SVG Management: Consider using D3.js or plain SVG with React refs for complex rendering
State Management: Zustand or Jotai for global state (song data, playback state)
Sustain Pedal Key: Recommend Space for sustain, or Shift as alternative
Grid Snapping: Use modulo operations with time quantization (e.g., snap to 1/16th notes)
Export Quality: Ensure SVG has proper viewBox for scaling, PNG at 2x resolution for print
Performance Scoring: Use timestamp comparisons with tolerance windows (¬±50-100ms for "good" timing)
